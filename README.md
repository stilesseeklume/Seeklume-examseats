# Seeklume Exam Room Scheduler

一个本地运行的高中大型考试排考工具。支持导入物理类、历史类学生成绩，生成主考、外语、小语种、三天拆分再选科考试/自习安排，并导出 Excel 材料。

## 特点

- 数据只保存在浏览器本机，不上传服务器
- 支持 `.xls`、`.xlsx`、`.csv`、`.tsv`、`.txt` 导入
- 自动识别常见成绩单字段
- 支持物理类先排、历史类后排
- 支持小语种外语单独考场
- 支持三天考试中化学、地理、政治、生物单科拆分
- 导出班主任表、考场信息表、门牌人数表、管理表、校验报告
- 校验漏排、重复、容量、混场、教室时段冲突

## 近期更新

- 预览区按“纸张”工作，能逐张选 sheet、逐张调方向和字号
- 考试时间支持粘贴通知文字或上传截图，再确认写入
- 门牌人数表会显示两天组合下的混排明细，方便看清每个考场装了什么
- 默认材料里去掉了重复的考务汇总，减少导出混乱

## 本地运行

```bash
npm install
npm run dev
```

打开终端提示的本地地址，一般是：

```text
http://127.0.0.1:5173/
```

## 验证

```bash
npm run test:rules
npm run build
```

测试脚本使用公开的模拟数据，不包含真实学生信息。

## 隐私说明

本项目不包含真实成绩单、导出材料或个人隐私数据。`.gitignore` 默认忽略 Excel、CSV、ZIP、构建产物和本地依赖。

真实考试数据请只保存在本机，不要提交到 GitHub。
